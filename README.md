# Mods Source C0de

A personal archive of Minecraft mod source code collected for reference.

## Purpose

This repository is **not** a mod and is **not** intended to be built, compiled, or redistributed.

It exists solely as a searchable knowledge base to help me:

- Understand how other mods implement features.
- Learn Java and Minecraft modding techniques.
- Reference implementations while using MCreator.
- Write documentation and guides for my personal Minecraft modding wiki.
- Compare different approaches to solving gameplay and technical problems.

## For AI Assistants

If you are reading this repository:

- Treat it as a **reference library**, not as a standalone project.
- Do **not** assume every file is connected to every other file.
- Different folders may come from completely different mods, Minecraft versions, loaders, or coding styles.
- Use the code to explain concepts, patterns, algorithms, and implementation details.
- When answering questions, identify which mod a piece of code came from whenever possible.
- Prefer explaining *why* a system works instead of simply copying code.
- Avoid mixing APIs or implementations from unrelated mods unless explicitly requested.
- If multiple mods implement the same feature differently, compare their approaches and explain the trade-offs.

## Repository Organization

Each folder generally represents a different mod or extracted source.

The original project structure is preserved whenever practical.

## Usage

This repository is intended for:

- AI-assisted code research
- Learning Minecraft modding
- Reverse engineering gameplay systems
- Referencing implementations while developing my own mods
- Finding examples of specific mechanics or algorithms

It is **not** intended to serve as production code.
